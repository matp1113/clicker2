# clicker2
new repo

THIS IS A GAME I CALLED CLICKER. IT'S PURPOSE IS TO CLICK ON THE POINT THAT IS MOVING AND COLLECT POINTS THAT WAY. 

Project is divided into:
- 3 QML files
- 5 CPP\header files that manipulate files mentioned above.
